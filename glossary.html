# Term Definitions
1 Countable Infinity A type of infinity that can be matched with natural numbers (e.g., integers).
2 Uncountable Infinity A larger type of infinity that cannot be matched one-to-one with natural numbers (e.g., real numbers).
3 Proof of the Successor A mathematical proof showing that each natural number has a unique next (successor) number.
4 Irrational Numbers Numbers that cannot be expressed as a ratio of two integers, with non-repeating, non-terminating decimal expansions (e.g., √2, π).
5 Dedekind 1872: Continuity and Irrational Numbers Establishes foundational concepts for set theory and defines irrational numbers rigorously.
6 Dedekind 1888: What Are Numbers and What They Should Be Introduces the idea of defining numbers through the successor function, forming the basis of set theory with S(n) = n + 1.
7 Principia Mathematica (1910) A comprehensive work by Whitehead and Russell that formalizes mathematical logic using set theory and builds upon Dedekind's ideas.
8 Skolem's Paradox (1921) Demonstrates that countable models can represent uncountable sets, contributing to model theory and first-order logic.
9 Completeness Theorem (1930) A theorem proving that any valid formula in first-order logic can be proven in a formal system, foundational to proof theory and computability.
10 Incompleteness Theorem (1931) Gödel's theorem stating that any sufficiently complex formal system contains true statements that cannot be proven within the system itself.
11 Lambda Calculus (1934) A formal system created by Alonzo Church that provides the basis for programming languages and computability theory.
12 Turing Machine (1936) Alan Turing's abstract machine that formalizes computation and proves there are problems that cannot be solved algorithmically (e.g., the halting problem).
13 Halting Problem (1936) The problem of determining whether a computer program will halt or run forever, shown by Turing to be unsolvable by any algorithm.
14 Von Neumann Architecture (1940) A computer architecture that separates program instructions from data storage, forming the foundation for modern computer design.
15 Fortran (1960) One of the earliest high-level programming languages, used for scientific and engineering computations.
16 Lisp (1960) A programming language based on lambda calculus, influential in artificial intelligence and functional programming.
17 Formal Logic The use of symbolic logic to express and prove mathematical statements in a structured manner.
18 P The class of problems that can be solved in polynomial time by a deterministic machine.
19 NP The class of problems for which a solution can be verified in polynomial time, but finding the solution may be challenging.
20 P=NP / P!=NP A major unsolved problem in computer science that asks whether every problem whose solution can be verified quickly can also be solved quickly.
21 NP-Completeness (1971) A concept introduced by Stephen Cook to classify computational problems based on their complexity and solvability.
22 Automated Theorem Proving (1980s) Development of algorithms to have computers check mathematical proofs, leading to advances in computer science.
23 Four Color Theorem (2005) A computer-assisted proof showing that any map can be colored with four colors without adjacent regions sharing the same color.
24 Proof of Four Color Theorem The use of computational methods to prove the longstanding mathematical problem, completed in 2005.
25 Large LLMs (2017-2024) Artificial neural networks designed for processing and generating text and visual data, which have become mainstream.
26 Quantum Computing (2024+) Computing devices that use quantum bits, potentially providing exponential speedup over classical computers, although still in development.
